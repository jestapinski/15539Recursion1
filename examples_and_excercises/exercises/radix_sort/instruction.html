The Algorithm:

 

Radix Sort is a non-comparison sorting algorithm, meaning it sorts a list without comparing the elements to one another.  How does it do that?  By going digit-by-digit!  The first iteration of Radix Sort orders numbers by their ones digit.  The second iteration orders them by their tens digit (note that the tens digit on a one-digit number is 0.)  At any time, if there are numbers with the same i-th digit, radix sort puts them in the order that the occur in the list.  Radix sort terminates after k iterations, where k is the number of digits in the largest number of the list.

 

How To Use:

This interactive excercise involves answering questions by writing numbers in a text field, and placing elements into the appropriate buckets.  There are three ways to place elements into the correct buckets:

  1. Dragging the element into a bucket,
  2. Clicking directly on the bucket,
  3. Pressing the key corresponding to the bucket (i.e. pressing "0" moves the element to the 0 bucket.)

Note that you can only move one element at a time, the enabled one (this is colored pink, whereas the rest are grey.)  This is because the algorithm goes through the elements of the list in order, hence you have to also.  Further note that the digit the algorithm is currently sorting by is highlighted in red, unless the number is too small to have that digit.

 

Grading:

In quiz mode, there are 22 points total:

  • How many iterations question: 2 points
  • Moving each element to the appropriate bucket: 1 point, 16 points total (0.5 points partial credit available if you place it in the wrong bucket but answer the followup question correctly.)
  • What index will this element move to question: 2 points each, 4 points total.
